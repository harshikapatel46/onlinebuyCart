# onlinebuyCart
A luxury editorial product card component built with HTML &amp; CSS — hover effects, color swatches, wishlist toggle, and animated interactions.
## Features
- Hover lift animation with image zoom
- "Quick Add" bar slides up on hover
- Clickable color swatches with live label update
- Wishlist heart toggle
- Sale / New badge variants
- Stock status indicator
- Fully responsive
- Zero JavaScript dependencies (minimal JS for swatch interaction only)
